# README - Farewell for now (December 2020)

I started playing this game again during my 2020 Cannon Beach (Oregon) vacation. I have really enjoyed getting back into this game. I made the goal for myself to get daily screenshots, and to max out my score, even if it means managing multiple phones at once.

I am getting close to reaching this goal, and am expected to reach it by the end of December 2020.

I am really going to miss this game. It has been really fun lately, and my admiration for the game stretches far beyond simple nostalgia. I admire every pixel this game has shone onto my screen. This game is a pure work of art, a jewel if you will call it.

I will have to play this game again in a virtual machine in the future. It has been fun.
